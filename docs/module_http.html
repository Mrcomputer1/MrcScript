HTTP

-m action:
Starts web server in current folder
-m args:
--port=(port number) - Default: 80, Required: No
--folder=(folder) - Default: "", Required: No

Contents:
init
addPage
run

[ ['http', 'init'], 'port' ]

Prepares the HTTP module

port: What port to run the web server on Required

[ ['http', 'addPage'], 'page path', 'page content' ]

Registers a page

page path: What is the file path Required
page content: What is content of the page (HTML) Required

[ ['http', 'run'] ]

Runs the server
